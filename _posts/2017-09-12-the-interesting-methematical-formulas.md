## 复利的计算公式

e = (1 + 1/n)^n

when n = ∞: e = 2.718281828459045

FV = PV * (1 + r/n)^n

其中：

{% raw %}
	FV = 终值

	PV = 现值

	r = 名义利率（例如，年利率）

	n = 期数

	when n = ∞: （1 + r/n)^n = e^r

		意味着，FV = PV * e^r
{% endraw %}
